# moapro.jp

Corporate website for **MOA Pro** — a live stream management company operating in partnership with BIGO LIVE.

Built solo from initial brief to final deployment, handling everything from UX design to implementation based on client feedback and iterative revisions.

🌐 **Live site:** [moapro.jp](https://moapro.jp)

---

## 📋 Project Overview

MOA Pro approached me with a vague description of what they wanted. I took ownership of the design direction, presented initial concepts, and iterated based on client feedback until final sign-off. All design and development work was handled by a single developer (me).

This was a real client project with actual stakeholder communication, revision cycles, and a live production deployment.

---

## ✨ Features

- 🎨 **Custom design** — Designed from scratch based on client brief, no template used
- 📄 **Full corporate site** — Company overview, services, and about pages
- 📬 **お問い合わせフォーム (Contact form)** — Fully functional inquiry form
- 📱 **Responsive design** — Optimized for mobile and desktop
- ⚡ **Performance optimized** — Fast load times via Next.js and Cloudflare

## 🚀 Getting Started

### Prerequisites

- Node.js 18+

### Installation

```bash
git clone https://github.com/yourusername/moapro.git
cd moapro
npm install
```

### Run Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎨 Design Process

This project involved a full design-to-delivery workflow:

1. **Discovery** — Initial brief from client with loose direction on desired look and feel
2. **Design** — Created full site design independently, balancing client brand expectations with modern web standards
3. **Implementation** — Built out design in code with responsive layout
4. **Feedback & Iteration** — Incorporated client revisions across multiple rounds
5. **Deployment** — Final delivery and deployment to production
